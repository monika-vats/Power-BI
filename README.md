This project showcases real-time data processing and dashboard development using Weather API integration with Power BI. The solution includes structured data modeling, DAX calculations, KPI tracking, and modern dashboard design principles.

---

### Core Capabilities
- Integration of live REST API data into Power BI
- Transformation and cleaning using Power Query
- DAX-based measure creation for weather and air quality insights
- Interactive visuals with user-friendly navigation and clean UI structure

---

### Dashboard Components
- Current weather temperature and condition
- 7-day forecast trend visualization
- Air Quality Index with pollutant-level breakdown (PM10, PM2.5, SO‚ÇÇ, CO, NO‚ÇÇ, O‚ÇÉ)
- Visibility, wind speed, humidity, and UV levels
- Sunrise and sunset time indicators
- Rain probability analysis using comparative bar visuals

---

### Tools & Frameworks
- Power BI Desktop
- Weather API (REST)
- Power Query M Language
- DAX (Data Analysis Expressions)

---

### Outcome
This dashboard supports quick decision-making, improves weather situation awareness, and demonstrates advanced reporting capability using external API data sources.
